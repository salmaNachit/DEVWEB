Chatbot Dev-Web

chatbot dev-web est un chatbot de définitions de langages de programmation est un outil pour aider les utilisateurs à comprendre les différents langages de programmation utilisés dans le développement web. Ce chatbot utilise une base de données riche en informations pour fournir des définitions claires des différents langages de programmation. Les utilisateurs peuvent poser des questions simples telles que "Qu'est-ce que le HTML?" ,ou seulement le nom de la language.

    Notre chatbot donne :

  • Definition de Devloppement web ,la partie back-end et fron-end front-end
  • Definition de chaque language de devloppement web
  • les fremework de chaque language et leur deffinition
  • les logicieles pour executer chaque language
  • pouvez poser votre question en anglait
